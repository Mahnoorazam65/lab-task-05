Task 2: Float-Based Magazine Layout

Floating Image on the Left

This article demonstrates how text wraps around an image floated to the left. The float property lets text flow around the image, similar to how newspapers or magazines align content.

Margins create space around the image, while clear ensures elements below start beneath floats.

Floating Image on the Right

In this example, the image is floated to the right so the text wraps on the left. Alternating float directions gives a balanced magazine feel.

To prevent collapsing issues, the parent uses overflow: hidden to contain floated elements.

Sidebar

The sidebar is floated to the right to simulate a second column, similar to the side sections of real magazine pages.
